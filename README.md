# Desafio Técnico - Frontend

## Rodando a API

Uma API de exemplo foi disponibilizada na pasta BACK.

Para rodá-la, faça:

```console
> cd BACK
> npm install
> npm run server
```

Ela responderá na porta 8080.

## Rodando o Front-end

```console
> cd FRONT
> npm install
> npm start
```

## OBS: 
- versão Node utilizada v14.18.1
- tive que alterar a porta no BACK para 8080
